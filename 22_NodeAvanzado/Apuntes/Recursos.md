## standard

Nos sirve para que escribir nuestro código de manera estándar

### Instalación
```
npm i -D standar
```

### Creamos un script
```json
"scripts": {
    "lint": "standard",
}
```

### Correrlo
```
npm run lint
```

Si tenemos errores los podemos arreglar con
```
npm run lint -- --fix
```
